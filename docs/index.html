Just a simple clock using jQuery library

First of all, it's just made for fun, and I'm still working on it(and perhaps I'll make a canvas version?).

Instructions:

First, load the jQuery library.

<script type="text/javascript" src="http://code.jquery.com/jquery-latest.min.js"></script>

And then, load the script and stylesheet.

<script type="text/javascript" src="clock.js"></script>
<link type="text/css" rel="stylesheet" href="clock.css">

Let's make it simple..

$('.clock01').clock();

How about another fancy style?

**Just add 'fancy' to the clock wrap.
$('.fancy').clock();

Here are the default values:

radius: 'auto', Highly recommend that use 'auto' if you are building a clock
tickType: '', Could be 'roman' / 'arabic' / 'zodiac' / 'chineseZodiac', or set to 'custom' if you want to use custom ticks.
rotateProps: true, Rotate the ticks or not.
responsive: true, Resize the clock if window size changed.
ticks: [] Custom ticks(if tickType is set to 'custom'), starts from 12.

And more... Here are some magic...

**This one is rotated.

**DO NOT use 'wheel' as the class if you don't want any trouble.
$('.wheel01').wheel({
    type: 'plate',
    props: 10
});

$('.wheel02').wheel({
    type: 'ring',
    props: 15
});

Here are the default values:

type: 'ring', Set to ring/plate as you want.
radius: 'auto', Set the size of the ring/plate.
**transform: translateY(-50%) will be applied to adjust the position if the type is set to 'plate', which would make the wheel bigger than your box.
props: 6, Set the number of props you want to generate.
wrapClass: 'wheel-wrap', Custom wrap class.
propClass: 'wheel-prop', Custom prop class.
rotateProps: true, Actually, it will rotate the props, not the ticks.
responsive: true Actually, it will resize the ring/plate, not the clock.
